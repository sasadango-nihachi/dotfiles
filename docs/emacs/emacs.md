# emacsについて


# 全体の移動
- C-v（Crtl ＋ v）で画面が下に移動する
- M-c（option + v）でもどるみたい
    - Escを押してからvでも同じことができる
    - Escもメタキーなのか
- C-lでカーソル位置の文字が画面の真ん中に移動した。
    - もう一度押すと上、もう一度押すと下に文字の位置が移動する

※MetaはWindowsではAlt,MacではOptionに該当、SuperはMacではcommandでwindowsには存在しない
